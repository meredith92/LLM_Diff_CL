Project Name

Language-Guided Diffusion for Domain Adaptive Segmentation (Industrial AI Demo)

🚀 Overview

This project focuses on domain adaptive segmentation for industrial scenarios (e.g., PCB conductor inspection), integrating:

🎯 Diffusion models for generative prior learning
🧠 LLM-based reasoning for structural judgment
🖼️ Segmentation models (EdgeSAM / lightweight models) for perception
⚡ Edge deployment (Jetson / Raspberry Pi + Hailo)

The goal is to build an end-to-end industrial AI pipeline that supports:

Perception → Generation → Reasoning → Decision

🧩 Key Features
✅ Domain adaptation with labeled + unlabeled data
✅ Diffusion-based structure-aware prior modeling
✅ Lightweight segmentation for edge inference
✅ Rule-based + LLM-based decision module
✅ Real-time deployment on embedded devices
