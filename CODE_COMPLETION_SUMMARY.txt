1. tools/uncertainty_analysis.py (560行)
   ────────────────────────────────────
   功能:
   - Diffusion uncertainty分析框架
   - 时间步不确定性计算 (多步denoising的方差)
   - 空间不确定性映射 (逐像素entropy)
   - 不确定性与错误的相关性分析
   - 域适应不确定性变化
   
   主要类:
   - DiffusionUncertaintyAnalyzer
   
   用法:
   python tools/uncertainty_analysis.py \
     --dataset-dir data/predictions \
     --output-dir work_dirs/uncertainty_analysis \
     --plot --save-maps


2. tools/ablation_study.py (480行)
   ─────────────────────────────────
   功能:
   - 系统的消融研究框架
   - 评估各组件的贡献:
     * Diffusion先验
     * EMA正则化
     * LLM引导
     * 规则拓扑
     * 完整模型
   - 自动可视化对比
   
   主要类:
   - AblationStudy
   
   用法:
   python tools/ablation_study.py \
     --config projects/pcb_conductor/configs/segformer_mt_vb.py \
     --work-dir work_dirs/ablations \
     --plot


3. tools/structure_analysis.py (650行)
   ────────────────────────────────────
   功能:
   - PCB导线引脚的拓扑分析
   - 引脚检测与度量
   - 结构质量指标:
     * 引脚计数一致性
     * 对齐评分
     * 均匀性评分
     * 拓扑违规率
   - 域特定结构洞察
   
   主要类:
   - StructureAnalyzer, Pin
   
   用法:
   python tools/structure_analysis.py \
     --pred-dir work_dirs/predictions \
     --gt-dir data/ground_truth \
     --output-dir work_dirs/structure_analysis \
     --visualize --plot


4. tools/train_continual.py (360行)
   ─────────────────────────────────
   功能:
   - 完整的端到端连续学习实验管道
   - 4阶段编排:
     1. Diffusion预训练 (Domain A标签)
     2. 分割模型训练 (Domain A)
     3. 两个域评估
     4. 连续微调 (Domain B)
   
   主要类:
   - ContinualExperimentOrchestrator
   
   用法:
   python tools/train_continual.py \
     --base-work-dir work_dirs/continual_experiment \
     --mask-root data/masks/domain_a

  ├── get_contribution(full_model, baseline, metric) -> Dict
  ├── save(output_file)
  └── plot_ablation_results(output_path, metric)
